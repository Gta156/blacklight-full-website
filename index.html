How NBT to Raw Works

This tool reads text files (like those generated by the Raw-to-NBT tool or some .mcstructure files) and uses pattern matching to find lines resembling commands (e.g., `"cmd_line": "..."` or `setblock ...`). It cleans up escape characters and presents the raw commands. It cannot parse true binary NBT. Use the filter option to only display `setblock` and `fill` commands.
